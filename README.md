# CommandLab 🛠️

**CommandLab** is a premium cybersecurity command generator designed for penetration testers and security professionals. It simplifies the process of generating complex commands for popular tools like `nmap`, `sqlmap`, `gobuster`, and more.

## 🚀 Features

- **Instant Generation**: Real-time command generation as you type.
- **Mobile Optimized**: Fully responsive design for use on the go.
- **Privacy Focused**: No database or tracking—your inputs stay in your browser.
- **Premium UI**: Sleek dark mode with high-performance animations.
- **Smart Navigation**: Automatically resets scroll on navigation for seamless use.

## 🛠️ Tech Stack

- **React 19**
- **TypeScript**
- **Vite**
- **Lucide React** (Icons)
- **Vanilla CSS** (Custom Design System)

## 📦 Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🌐 Deployment

This project is configured for deployment on **Vercel**.
